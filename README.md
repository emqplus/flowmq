# FlowMQ

FlowMQ is an experimental MQTT broker written in C++ with cursor.

## Prerequisites

- C++ compiler with C++17 support
- CMake 3.10 or higher

## Building

```bash
mkdir build && cd build
cmake ..
make
```

## Run

```bash
./flowmq
```
